# timer
Simple countdown timer with transparency, python + pyqt5. Option to toggle "always on top" mode.

	git clone https://github.com/VS-W/timer
	cd timer
	python3 -m venv .
	source bin/activate
	pip3 install -r requirements.txt
	python3 timer.py &
